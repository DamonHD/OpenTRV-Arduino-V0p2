This is a simple collection of copies of CADSoft-Eagle-compatible libraries (http://www.cadsoft.de/)
that we've created for OpenTRV (and related projects) and which we'd like to share.

The normal Apache licensing applies.

Here is a summary of what's in some of them:


Name            Content
====            =======
Arduino-AVR     ATmega328 (+164, etc) -P-PU IC package.
FDTI            6-pin connector for FDTI USB/serial connection for AVR.
screwterminal   Simple through-the-hole screw terminal connector.
shorts          Simple short(s) to allow two logically-separate nets to connect.
TMP112          Package for TMP112 (and TMP102) SMD temperature sensor.
